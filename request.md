## 三、C语言基础知识
**1. 核心概念**
* **文件指针 (`FILE*`)**: 用于标识和操作文件的指针，通过 `fopen` 获取，`fclose` 释放。
* **逐字符读取 (`fgetc`)**: 从文件中一个字符一个字符地读取，是文本分析的基础操作。
* **状态机思想**: 通过“标记变量”（如 `in_word`）记录当前状态（是否在单词中），根据输入字符切换状态，实现单词统计。
* **字符判断函数 (`ctype.h`)**: `isalpha` (判断字母)、`isdigit` (判断数字)、`tolower` (转小写)等，用于字符分类和处理。
* **边界情况处理**: 空文件、文件最后一行无换行符、单词含数字（如“C99”）等特殊场景的处理逻辑。

## 五、基础版代码核心分析
**1. 核心变量定义**
```c
FILE *fp;                                // 文件指针
char filename[] = "text_test.txt";       // 固定分析的文件名
int ch;                                  // 存储当前读取的字符
int chars = 0;                           // 字符总数(包括空格、标点、换行)
int words = 0;                           // 单词总数
int lines = 0;                           // 行数
int in_word = 0;                         // 状态机标记:1=在单词中,0=不在单词中
```

**2. 文件打开与错误处理**
```c
fp = fopen(filename, "r"); // 以“只读”模式打开文件
if (fp == NULL) {
    // 打开失败处理
    printf("错误:无法打开文件 %s\n", filename);
    return 1;
}
```
*分析:* `fopen` 的第二个参数 `"r"` 表示只读；必须检查 `fp` 是否为 `NULL`，避免程序崩溃。

**3. 逐字符读取与统计（核心逻辑）**
```c
while ((ch = fgetc(fp)) != EOF) { // EOF是文件结束标志
    chars++;                      // 每读一个字符,字符数+1(包括空格、换行)
    
    if (ch == '\n') {
        lines++;                  // 遇到换行符,行数+1
    }
    
    // 状态机:单词统计
    if (isalpha(ch)) {            // 当前字符是字母
        if (!in_word) {           // 之前不在单词中→新单词开始
            in_word = 1;
            words++;              // 单词数+1
        }
    } else {                      // 当前字符不是字母→单词结束
        in_word = 0;
    }
}
```
*重点分析: 状态机思想*
* 用 `in_word` 标记“当前是否在单词中”；
* 当遇到字母且 `in_word=0` 时，说明新单词开始，单词数+1并将 `in_word` 设为 1；
* 当遇到非字母字符时，说明单词结束，将 `in_word` 设为 0；
* 这种方法避免了“连续空格导致重复计数”的问题，是文本处理的经典思想。

**4. 边界情况处理：最后一行无换行符**
```c
if (chars > 0 && lines == 0) {
    lines = 1; // 文件非空但无换行符→整个文件算1行
}
```
*分析:* 如果文件只有一行且没有换行符，`lines` 会是0，需要手动修正为1。

**5. 文件关闭与结果输出**
```c
fclose(fp); // 必须关闭文件,避免资源泄漏

printf("文件 %s 统计结果:\n", filename);
printf("字符数:%d\n", chars);
printf("单词数:%d\n", words);
printf("行数:%d\n", lines);
```

**任务2.1：单词频率统计（10分）【已完成】**
* **功能要求：** 统计每个单词出现的次数，不区分大小写（如“Hello”和“hello”算同一个单词）。
* **实现提示：**
    1. 定义结构体存储单词和频率：
    ```c
    #define MAX_WORDS 100    // 最多统计100个不同单词
    #define MAX_WORD_LEN 50  // 单词最大长度
    
    typedef struct {
        char word[MAX_WORD_LEN]; // 单词(转小写存储)
        int count;               // 出现次数
    } WordFreq;
    ```
    2. 实现函数 `void compute_word_freq(FILE *fp, WordFreq *freq, int *total_words)`:
        * 逐字符读取文件，提取单词（用状态机），并转小写；
        * 遍历 `freq` 数组，若单词已存在则 `count++`，否则添加新单词；
    3. 在 `main` 中调用该函数，格式化输出词频（用 `%-20s` 对齐）。

**任务2.2：字母频率统计（10分）【已完成】**
* **功能要求：** 统计每个英文字母(a-z,不区分大小写)出现的次数和频率(次数/总字母数)。
* **实现提示：**
    1. 定义数组存储字母频率: `int letter_freq[26] = {0};` (下标0对应a, 1对应b, ..., 25对应z)。
    2. 逐字符读取文件，用 `tolower(ch)` 转小写，若 `isalpha(ch)` 为真，则 `letter_freq[ch - 'a']++`。
    3. 计算总字母数，遍历数组输出每个字母的次数和频率（保留2位小数）。

**任务2.3：最长单词查找（10分）【已完成】**
* **功能要求：** 找出文本中最长的单词，若有多个长度相同的最长单词，输出第一个。
* **实现提示：**
    1. 定义变量存储最长单词: `char longest_word[MAX_WORD_LEN] = ""; int max_len = 0;`。
    2. 逐字符读取文件，提取当前单词，用 `strlen` 获取长度；
    3. 若当前单词长度 > `max_len`，则更新 `max_len` 和 `longest_word`。
    4. 输出最长单词及其长度。
